# CyberSigma

Detection rules in [Sigma format](https://github.com/SigmaHQ/sigma) for modern threats. Generated and maintained by AgentForge from live threat intelligence.

## Usage

Rules are standard Sigma YAML. Compile to your SIEM with [pySigma](https://github.com/SigmaHQ/pySigma) or [sigmac](https://github.com/SigmaHQ/sigma):

```bash
# Splunk
sigma convert -t splunk rules/netlogon_rce_cve_2026_41089.yml

# Elastic
sigma convert -t elasticsearch rules/weblogic_jndi_t3_cve_2024_21182.yml

# Wazuh (XML)
sigma convert -t wazuh rules/container_escape_linux_kernel.yml
```
